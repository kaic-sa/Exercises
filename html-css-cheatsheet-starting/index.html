CSS Position and Display

Position propeties

Property Function
static The element scroll along container. This is the default value
relative The element offset will be based on top, right, bottom and left attributes
absolute The element will be removed from normal flow of document, with no space created in the page layout
sticky The element will be removed from normal flow of document, with no space created in the page layout, but positioned relative to his parent

Display properties

Property Function
inline The element generates one or more inline element boxes that do not generate line breaks
block The element generates a block element box, generating line breaks
inline-block The element generates a block element box that will be flowed with surrounding content
